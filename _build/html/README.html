Splat¶

Splat is a functional language designed for a module coursework, in a couple of weeks.

Features¶

  • Syntax that is difficult to read.
  • Very limited function.
  • Functional language.

Installation¶

Splat is very easy to install. Just run:

make

And use with:

mysplinterpreter input.spl < stream
